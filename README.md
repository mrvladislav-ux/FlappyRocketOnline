## Features

* Arcade gameplay
* Multiple difficulty modes
  - Easy
  - Medium
  - Hard
  - Dynamic
* User registration & login
* JWT authentication
* Online leaderboard
* SQLite database with Entity Framework Core
* REST API

## Tech Stack

* C#
* Raylib-cs
* ASP.NET Core
* Entity Framework Core
* SQLite
* JWT
* BCrypt
